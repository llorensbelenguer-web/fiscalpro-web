# Plan de Estructura: Web de Consultoría Fiscal para Autónomos y PYMES

## 1. Mapa del Sitio Sugerido
- **Inicio (Home):** Propuesta de valor clara, servicios destacados y llamada a la acción (CTA) para una consulta.
- **Servicios:**
    - Asesoría para Autónomos (Alta, IVA, IRPF).
    - Gestión Fiscal para PYMES (Impuesto de Sociedades, Cuentas Anuales).
    - Planificación Fiscal y Optimización.
    - Auditoría y Recursos ante la AEAT.
- **Blog (Actualidad Fiscal):** Artículos sobre cambios en leyes, plazos de impuestos y consejos de ahorro.
- **Sobre mí / Nosotros:** Generar confianza (equipo, experiencia, valores).
- **Contacto:** Formulario de consulta, calendario para citas y ubicación/datos de contacto.
- **Legal:** Aviso Legal, Política de Privacidad y Cookies (Crítico en este sector).

## 2. Temas de Contenido Inicial (Estratégico para SEO)
1. **Guía de Alta en Autónomos:** Todo lo que necesitas saber antes de empezar.
2. **Calendario Fiscal 2026:** Fechas clave para el IVA e IRPF.
3. **Gastos Deducibles:** ¿Qué puedes desgravar realmente en tu actividad?
4. **Impuesto de Sociedades:** Novedades y beneficios para pequeñas empresas.

## 3. Stack Técnico Recomendado (Astro)
- **Framework:** Astro (v4+).
- **Estilos:** Tailwind CSS (para un diseño limpio y profesional).
- **Contenido:** Markdown o MDX para los artículos del blog.
- **Despliegue:** Netlify o Vercel (gratuito para este tipo de sitios).
- **Formularios:** Netlify Forms o Formspree.
